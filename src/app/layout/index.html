Note

  • SlicePipe is used to creates a new List or String containing a subset (slice) of the string or array

Syntax: array_or_string_expression | slice:start[:end]

  • array_or_string_expression is the string to slice
  • slice is the name of the pipe
  • start is the start position/index from where the slicing will start
  • end is the ending index/position in the array/string

Example

  • {{ collect }}

{{str}}[0:4]: '{{str | slice:0:4}}' - output is expected to be 'abcd'

{{str}}[4:0]: '{{str | slice:4:0}}' - output is expected to be ''

{{str}}[-4]: '{{str | slice:-4}}' - output is expected to be 'ghij'

{{str}}[-4:-2]: '{{str | slice:-4:-2}}' - output is expected to be 'gh'

{{str}}[-100]: '{{str | slice:-100}}' - output is expected to be 'abcdefghij'

{{str}}[100]: '{{str | slice:100}}' - output is expected to be ''
