Note:

  • There are different ways to bind CSS classes to NgClass that are using string, array and object
  • We can assign one or more than one CSS classes to HTML element using NgClass
  • When we use NgClass with object then adding and removing CSS classes are performed on the basis of retuned Boolean value of an expression. If the value of expression is true then respective CSS class will be added otherwise that CSS class will be removed from HTML element at run time
