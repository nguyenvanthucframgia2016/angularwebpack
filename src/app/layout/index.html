Note

  • Pipes helps us to format or transform data to display in our template
  • angular 2 built in pipes like currency pipe, date pipe, number pipe, percent pipe, decimal pipe, & slice pipe

Pipes demo

Unformatted Date: {{ today }}
Formatted Date: {{ today | date | uppercase }}

DatePipe

Syntax: date_expression | date[:format]
date_expression is a date object or a numbe
date is the name of the pipe
format is the date and time format string which indicates the format in which date/time components are displayed.

Example DatePipe

medium : {{today | date:'medium'}}

short : {{today | date:'short'}}

fullDate : {{today | date:'fullDate'}}

longDate : {{today | date:'longDate'}}

mediumDate : {{today | date:'mediumDate'}}

shortDate : {{today | date:'shortDate'}}

mediumTime : {{today | date:'mediumTime'}}

dd-MM-y : {{today | date:'dd-MM-y'}}

yy-MM-dd : {{today | date:'yy-MM-dd' }}

dd-MM-yy HH:mm : {{today | date:'dd-MM-yy HH:mm'}}

Reference: Angular DatePipe
